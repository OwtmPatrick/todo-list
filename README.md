# Web3 connect wallet

 Web3 connect wallet implementation. Available options are metamask, coinbase wallet and wallet connect

## Start

Install packages

```
npm install
```

Start dev server

```
npm run dev
```

## Build

```
npm run build
```

## Run linter

```
npm run lint
```

<code>--fix</code> flag avalilable as well:

```
npm run lint:fix
```
